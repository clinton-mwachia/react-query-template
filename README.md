# react-query-template
Learn React query with this project
To use this project

CLONE THE REPO
`git clone https://github.com/clinton-mwachia/react-query-template.git`

START THE SERVER
`json-server --watch db.json` (incase it doesn't work in terminal, use cmd)

START THE APP
`npm run dev`
